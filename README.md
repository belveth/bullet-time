# Getting Started
## build
```bash
$ ./gradlew build
```

## start server
```bash
$ java -Dspring.profiles.active=local -jar build/libs/bullet-time-x.x.x-SNAPSHOT.jar
```
- [localhost:3000](http://localhost:3000)
